About DQR

DQR is an opensource rust based qr code generator.


- Purpose

DQR Has the goal to be an local fast and bloat free qr code generator


- State

DQR is finished but may get more features

DQR is opensource and sourcecode may be viewed at github.

For bugs or issues and issue may be created at github.

DQR is under the MIT license.




Installation

DQR can be installed in 3 ways.


- MSI Installer

Using the .msi installer, DQR can be installed with admin privelages.


- EXE Installer

Using the .exe installer, DQR can be installed without admin privelages.

This method is recommended.


- Building from source

View the build from source guide on github.

This method is not recommended.


After you have followed the building steps.

Run you preferred installer with info above.




Usage

Put in an url or text in the url input after that click generate to see an preview and generator(You can do this directly) to export it to downloads



Deletion

Go to the settings app and find DQR and click Deinstall.


- Extra option EXE

Or assuming you choosed the default installation settings.

Go to %AppData%\Local\dqr.

And run uninstall.exe
