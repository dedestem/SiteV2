About DFinder

DFinder is an opensource rust based filebrowser.


- Purpose

DFinder has as purpose to be an FAST filebrowser.

DFinder can find files faster than the default windows filebrowser.


- State

DFinder is still in development and docs may be missing.

DFinder is opensource and sourcecode may be viewed at github.

For bugs or issues and issue may be created at github.

DFinder is under the MIT license.




Installation

DFinder can be installed in 3 ways.


- MSI Installer

Using the .msi installer, DFinder can be installed with admin privelages.

Using this method allows you to access all files.


- EXE Installer

Using the .exe installer, DFinder can be installed without admin privelages.

Using this method allows you to access user only users files.

This method is recommended.


- Building from source

View the build from source guide on github.

This method is not recommended.


After you have followed the building steps.

Run you preferred installer with info above.
