Placeholder

Docs > Placeholder API

Main

1) Random avatar

https://placeholder.davidnet.net/public

2) Random boy avatar

https://placeholder.davidnet.net/public/boy

3) Random girl avatar

https://placeholder.davidnet.net/public/girl

4) Unique avatar by ID

https://placeholder.davidnet.net/public/[ID]

Example: ID=60




5) Avatar based on username

General

https://placeholder.davidnet.net/public?username=[value]

Example: username=Jordan

Boy

https://placeholder.davidnet.net/public/boy?username=[value]

Example: username=Scott

Girl

https://placeholder.davidnet.net/public/girl?username=[value]

Example: username=Angela

Avatars with Initials From Names

https://placeholder.davidnet.net/public/girl?username=[value] - Scott+Wilson
